# AI vs. Human Bug Patterns in Competitive Programming

Empirical study comparing the bug patterns of AI-generated and human-written incorrect
solutions to the same CodeContests problems under one fixed taxonomy. See
`RESEARCH_EXECUTION_PLAN.md` for the full design and `repro/REPRODUCE.md` (added in
Phase I) for how to rebuild all artifacts. Single entry point: `python run.py <cmd>`.
